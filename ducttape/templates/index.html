About the Ducttape Project

The Ducttape Project is an open source game development environment consisting of an engine and an editor. The main goal of the project is to greatly ease the creation of modern games by doing most of the heavy lifting and architecture design upfront.

The name "Ducttape" was chosen because we build on the shoulders of giants (see engine) and provide the "duct tape" to transparently use them.
