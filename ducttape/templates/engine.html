Engine

The Ducttape Engine is a C++ open-source game engine using Ogre3D for graphics, SFML for network and audio and the Bullet Physics engine.

License

The Ducttape Engine and its components such as the editor as well as examples and tests are available under the LGPL3 license unless noted otherwise.

Goals

This is the list of goals the Ducttape Engine is made for, sorted by descending priority.

  1. Compatibility
  2. Maintainability
  3. Usability
  4. Performance
  5. Feature richness
  6. Extensibility
